Path Integrals

Path Approximation

A continuous path can be approximated by a finite number $N$ of straight line segments.

Path Calculus

In the limit as the $N \rightarrow \infty$ the path approximation becomes equivalent to the continuous path.

Alternative Path Postulate

Alternative paths are paths that have the same initial spacetime coordinates and the same final spacetime coordinates.

The resultant amplitude for alternative paths is the sum of the amplitudes for each alternative.

$ K = \sum_{i} K_i $

The resultant amplitude for sequential paths is multiplicative.

$ K_{CA} = K_{CB} K_{BA} $

Propagator Postulate

We use one coordinate $q$ to represent the position transverse to the initial-final axis and use time $t$ as a proxy for the longitudinal progress.

The amplitude $K$ for an infinitesimal path from $(q, t)$ to $(q + \Delta q, t + \Delta t)$ is assumed to be

$ \begin{eqnarray*} K & = & \langle q + \Delta q, t + \Delta t | e^{-i H \Delta t} |q, t \rangle \end{eqnarray*} $

Empirically, $H$ is determined to be the quantum-mechanical Hamiltonian operator.
