Lagrangian Formalism

Systems of Particles

The positions of $N$ particles each with position vector $\vec{r}_{i}$ where $i = 1 \ldots N$, along with their velocities, $\vec{v}_{i}$, are used to specify the configuration of the system.

The equations of motion that determine the evolution of the system are

$ \begin{eqnarray} m_{i} \frac{d \vec{v}_{i}}{d t} & = & \vec{F}_{i} \end{eqnarray} $

and

$ \begin{eqnarray} \frac{d \vec{r}_{i}}{d t} & = & \vec{v}_{i} \end{eqnarray} $

Notice that in this context, $i$ is an index that picks out a particular particle.

Configuration Space

Instead of labelling the system coordinates by particle index, we describe the configuration of the system as if it were a single particle with 3N position coordinates. The set $\{x_{i}\}$ defines a point in configuration space. Each point is a configuration of the system and the trajectory of the system is a curve. If there are N particles, each with 3 degrees of freedom then there are 3N coordinates to describe each point. Note that keeping track of the full configuration of the system also requres the set $\{ \dot{x}_{i} \}$. We will revisit this issue later when we describe phase space.

The Classical Action

Let $\bar{x}(t)$ be the trajectory of a macroscopic particle. We call this the classical trajectory.

There exists a quantity $S[x]$, called the action, which can be computed for an arbitrary path, $x(t)$.

$ \begin{eqnarray} S[x] & = & \int_{t_{i}}^{t_{f}} L(x(t), \dot{x}(t), t) dt \end{eqnarray} $

The function $L$, called the lagrangian, computes a value at each point along the path.

With just this defintion, $L$ could be any function and the value of $S$ for a given path is therefore arbitrary. We need a principle to constrain $L$.

We avoid defining $L$ as anything in particular, leaving the actual form of $L$ to be determined. Some authors define $L$ to be something and make the following principle a theorem.

Principle of Least (Stationary) Action

Along the classical path, $S$ an extremum with respect to a change in the path.

$ \begin{eqnarray} \delta S & = & S[\bar{x} + \delta x] - S[\bar{x}] = 0 \end{eqnarray} $

We can now crank the handle to see what this means for $L$. The technique we use comes from the Calculus of Variations.

Suppose we try a path that differs a bit from the classical trajectory

$ \begin{eqnarray} x(t) & = & \bar{x}(t) + \alpha \eta(t) \end{eqnarray} $

where $\alpha$ is a scalar that controls how much we deviate from the classical trajectory $\bar{x}$. Along with the conditions that $\eta(t_{i}) = \eta(t_{f}) = 0$, we deduce that the lagrangian must satisfy a second order differential equations.

Euler Lagrange Equations

$ \begin{eqnarray} \frac{d}{d t} \frac{\partial L}{\partial \dot{x_{i}}} & = & \frac{\partial L}{\partial x_{i}} \end{eqnarray} $

We will later show that these equations also hold for a general coordinate transformation. This will be powerful because it extends the Lagrangian formalism to non-inertial frames and allows constraints to be handled more easily.

Computing the Lagrangian

The form of the lagrangian is usually reverse engineered. Knowing the actual equations of motion, we guess a lagrangian and see if it produces the correct force law.

For a particle of mass $m$ subjected to a potential $V(x,t)$, the lagrangian is

$ \begin{eqnarray} L & = & \frac{1}{2} m \dot{x}^{2} - V(x,t) \end{eqnarray} $

Universality of the Lagrangian formalism

All the fundamental laws of physics can be written using the action principle

$ \begin{eqnarray} L & = & \sqrt g (R - \frac{1}{2} F_{\mu \nu} F^{\mu \nu} + \bar{\phi} \mathbb{D} \phi) \end{eqnarray} $

Coordinate System Invariance

Consider the coordinate transformation
$ \begin{eqnarray} q_{i} & = & q_{i}(x_{j}, t) \end{eqnarray} $

Since this is a coordinate transformation we must have
$ \begin{eqnarray} x_{j} & = & x_{j}(q_{i}, t) \end{eqnarray} $

Using the properties of partial derivatives under coordinate transformations we can show

$ \begin{eqnarray} \frac{d}{d t} \frac{\partial L}{\partial \dot{q_{i}}} & = & \frac{\partial L}{\partial q_{i}} \end{eqnarray} $

Generalized Momentum Conjugate to $q_{i}$

$ \begin{eqnarray} p_{i} & = & \frac{\partial L}{\partial \dot{q_{i}}} \end{eqnarray} $

Mechanical momentum (mass times velocity) is elementary mechanics.

Canonical momentum (derivative of Lagrangian wrt velocity) is advanced mechanics.

Cyclic Coordinates

Lagragian depends on velocity rather than a configuration coordinate.

Lagragian is independent of some generalized configuration coordinate.

Conjugate Momentum is conserved.
