Momentum

Single Particle Systems

Linear Momentum of a Particle

The linear momentum, $\Vector{p}$, of a particle with mass $m$ and velocity $\Vector{v}$ is defined to be

$\Vector{p} = m\Vector{v}$

Linear Momentum in DaVinci Python

from e3ga import * from units import * m = ScalarE3(5) * kilogram v = VectorE3(1,2,0) * meter / second p = m * v print p 

        
      

    

    

    
    

  


  

    

      

        
Angular Momentum of a Particle about a Point

      

      

        

          The moment of momentum of a particle about a given point is called the angular momentum about that point.
        

        

          If $\Vector{r}$ is the displacement from the point to the particle, and $\Vector{p}$ is the linear momentum of the particle, then the angular momentum, $L$, of the particle about the point is


          $L = \Vector{r} \wedge \Vector{p}$
        

      

    

  

  

    
Angular Momentum in DaVinci Python

    

      

        
from e3ga import *
from units import *

r = VectorE3(0,0,1) * meter
m = ScalarE3(5) * kilogram
v = VectorE3(1,2,0) * meter / second

p = m * v
L = r ^ p

print L
