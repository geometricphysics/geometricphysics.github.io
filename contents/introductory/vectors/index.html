Vectors

Why do we care about vectors?

The simple answer is that a vector can be used to model certain kinds of physical quantities. A more complete answer is that vectors and similar geometric objects can be used to model all physical quantities. It will make more sense to postpone defining what we mean by a geometric object for now, but we can say a vector is one example. As a final teaser, we will discover that nature's laws tend to be simplest when expressed correctly in terms of geometric quantities.

For a familiar example of a vector, the displacement (change in position), $\Delta \geometric{x}$, of an object from one point to a nearby point can be modeled as a vector.
