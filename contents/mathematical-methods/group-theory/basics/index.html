Group Theory Basics

Definition

A group $G$ is a set of elements $\{a,b,c,\ldots\}$ together with a binary composition law, called multiplication, which has the following properties:

  1. Closure. The composition of any two elements $a$ and $b$ in $G$, called the product and written $a b$, is itself an element of $G$.

  2. Associativity. The composition law is associative, i.e., for any elements $a, b, c \in G$, $(ab)c = a(bc)$.

  3. Identity. There exists an element, called the unit or identity and denoted by $e$, such that $ae = ea = a$ for every element $a$ in $G$.

  4. Inverses. Every element $a \in G$ has an inverse, denoted by $\Inverse{a}$, which is also in $G$, such that $\Inverse{a} a = a \Inverse{a} = e$.

Unique Identity Theorem

The identity element in a group $G$ is unique.

Cancellation Theorem

In a group $G$, the left and right cancellation laws hold, i.e., $ab = ac$ implies $b = c$. and $ba = ca$ imples $b = c$.

Notice that the cancellation does not require $a$ to be arbitrary element.

Unique Inverse Theorem

For each element $a$ in a group $G$, there is a unique element $b$ in $G$ such that $ab = ba = e$.

Rearrangement Theorem

If $G = \{e, g_1, \ldots, g_{n}\}$ then the set $G g_k = \{e g_k, g_1 g_k, \ldots, g_{n} g_k\}$ contains each element of $G$ exactly once.

The proof is by contradiction and uses the Cancellation Theorem.

Multiplication Tables

The composition law for a finite group can be represented as a multiplication table.

The Rearrangement Theorem says that the rows or columns of the tables can only contain each element once.

Interestingly, the multiplication table for the finite groups $\{e,a\}$ and $\{e,a,b\}$ are completely determined by the group definition and theorems and so the algebraic structures of groups with two or three elements are unique.

A group with four elements $\{e,a,b,c\}$ yields two distinct multiplication tables. One structure has all the elements squaring to $e$. The other has only one element squaring to $e$.

Coset

A left (right) coset is the set formed from a subgroup of $G$ where each element has been multiplied on the left (right).

Coset Common Element Theorem

Two cosets of a subgroup either contain exactly the same elements or else have no elements in common.

Proof. Let $H = \{e, h_1, \ldots, h_r\}$ be the common subgroup of $G$ used to create each coset, with $g_k$ being an element of $G$. Let $H g_1$ and $H g_2$ be the two right cosets. If the common element is $h_i g_1 = h_j g_2$ then

$ \begin{eqnarray*} g_1 \Inverse{g_2} & = & \Inverse{h_i} h_j. \end{eqnarray*} $

But $\Inverse{h_i} h_j$ is an element of $H$, and so the group $H \Inverse{h_i} h_j = H g_1 \Inverse{g_2}$ is a rearrangement of $H$.

Therefore the elements of $H g_2$ are identical to those of

$ \begin{eqnarray*} (H g_1 \Inverse{g_2}) g_2 & = & H g_1 (\Inverse{g_2} g_2) = H g_1. \QED \end{eqnarray*} $

Lagrange's (Order Divisor) Theorem

The order of a subgroup $H$ of a finite group $G$ is a divisor of the order of $G$, i.e., $\Norm{H}$ divides $\Norm{G}$.

Proof.

Let $G = \{e, g_1, \ldots, g_n\}$ be a group with order $\OrderOfGroup{G}$.

Let $H = \{e, h_1, \ldots, h_r\}$ be a subgroup of $G$ with order $\OrderOfGroup{H}$.

Construct cosets from the subgroup $H$ and every element in $G$.

$H$ contains $e$, $e g_k = g_k e = g_k$, and so every element in $G$ must appear in at least one coset.

But the cosets produced may not be distinct because any cosets with elements in common have all elements the same.

Let $k \in \Integer$ be the number of distinct cosets, which is called the index of the subgroup $H$.

(The index of a subgroup is the number of distinct cosets that can be created from it).

All cosets formed from the same subgroup have the same number of elements, $\OrderOfGroup{H}$.

So we must have $k \OrderOfGroup{H} = \OrderOfGroup{G}. \QED$

Lagrange's Theorem restricts the allowable orders of the subgroups of a given group. The converse is not generally valid, i.e., the orders of subgroups of $G$ need not span the divisors of $\OrderOfGroup{G}$.
