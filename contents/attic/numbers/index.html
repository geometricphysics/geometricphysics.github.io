Numbers

Natural Numbers

$\{1,2,3,\ldots\}$

Real Axis

We will assume the concept of a real number as represented by points on a line.

Point Set

A set of points (or real numbers) is called a one-dimensional point set.

Countability

A set is said to be countable if its elements can be placed in 1-1 correspondence with the natural numbers.

Variable

A symbol $x$ which can represent any number or point of a set is called a variable.

Constants

Fixed numbers on the real axis are called constants.

Interval

A set of points $x$ such that $a ≤ x ≤ b$ is called a closed interval and is denoted by $[a,b]$.

$[a,b] = \{x \in \mathbb{R}: a ≤ x ≤ b \}$

A set of points $x$ such that $a < x < b$ is called an open interval and is denoted by $(a,b)$.

$(a,b) = \{x \in \mathbb{R}: a < x < b \}$

Neighborhood

The set of all points $x$ such that $|x-a| < \delta$, where $\delta > 0$ is called a $\delta$ neighborhood of the point a.

The set of all points $x$ such that $0 < |x-a| < \delta$, where $\delta > 0$ is called a deleted $\delta$ neighborhood of the point a.

Limit Point

A limit point of a set of numbers is a number $l$ such that every deleted $\delta$ neighborhood of $l$ contains members of the set.

For any $\delta$, there exists $x ≠ l$ such that $|x - l| < \delta$.

Closed Set

A set that contains all its limit points is called a closed set.

Bounds

If for all numbers $x$ of a set there exists a number $M$ such that $x ≤ M$ then the set is bounded above and $M$ is called the upper bound.

If for all numbers $x$ of a set there exists a number $m$ such that $x ≥ m$ then the set is bounded below and $m$ is called the lower bound.

If a set is bounded above and below, the set is bounded.

If $M_{L}$ is an upper bound for set and there is at least one member which is greater than $M_{L}-\epsilon$ for every $\epsilon > 0$ then $M_{L}$ is called the least upper bound of the set.

If $m_{G}$ is a lower bound for set and there is at least one member which is less than $m_{L}+\epsilon$ for every $\epsilon > 0$ then $m_{G}$ is called the greatest lower bound of the set.

Weierstrass-Bolzano Theorem

Every bounded infinite set has at least one limit point.
